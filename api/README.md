# Golden Raspberry Awards API

## Installation

```bash
npm install
```

## Database Migration

### Antes de rodar o projeto, execute as migrations para criar as tabelas do banco de dados:

```bash
npm run knex -- migrate:latest
```

## Development

```bash
npm run start:dev
```

## Production

```bash
npm run build
npm run start
```

## Tests

```bash
npm run test
```
